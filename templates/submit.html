HIPC Validation Demo

Submit an Excel (.xslx) file for validation:

This tool will look in the spreadsheet for a column with the title 'Virus Strain'. It will check each value in that column and write an Excel file that highlights and offers suggestions for those values:

  1. green: exact match to the NCBI Taxonomy scientific name for a virus
  2. blue: automatic replacement of an unambiguous match with the exact scientific name for a virus
  3. orange: suggestion for manual replacement
  4. red: not recognized as a name for a virus

See the source code: https://github.com/jamesaoverton/hipc-validation
